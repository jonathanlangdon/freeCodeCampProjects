Variables

In JavaScript, variables are used to store and manipulate data. They can be declared using the var, let, or const keywords, and can hold a variety of data types, such as strings, numbers, and booleans. Understanding how to work with variables is crucial to building effective JavaScript applications. In this section, we will explore the basics of declaring and initializing variables, naming conventions, variable types, scope, and more. By the end of this section, you should have a solid understanding of how to use variables in your JavaScript code.

Here is an example of the two main types of variables that modern Javascript

const is used to declare a variable whose value will not change. This is the preferred default declaration.

        const yearIndependence = 1776;

While let is used to declare a variable that will change later on.

        let temperature = 42;
        temperature = 39;

Data Types

In JavaScript, data types define the type of value that can be stored in a variable. Understanding data types is crucial to building effective JavaScript applications. In this section, we will explore the different data types in JavaScript, including strings, numbers, booleans, null, undefined, and objects. We will also discuss how to work with these data types and how to convert between them. By the end of this section, you should have a solid understanding of how to use data types in your JavaScript code.

Data Types:

  • String: A sequence of characters that can be enclosed in single or double quotes.
  • Number: A numeric value that can be an integer, a floating-point number, or a value represented in exponential notation.
  • Boolean: A value that can be either true or false.
  • Null: A special value that represents nothing, empty, or non-existent.
  • Undefined: A value that represents a variable that has been declared but has not been assigned a value.
  • Object: A complex data type that can hold multiple key-value pairs, and can be used to represent more complex data structures.

    • Here is an example of an object:

                const car = {
              make: "Toyota",
              model: "Camry",
              year: 2022,
              isAutomatic: true,
              color: "blue"
            };

    Type Conversion

    In JavaScript, type conversion is the process of converting a value from one data type to another. This is an important concept to understand because it allows you to work with different data types in a flexible and dynamic way. In this section, we will explore the basics of type conversion in JavaScript, including the difference between implicit and explicit conversion, how to convert between different data types, and common use cases for type conversion. By the end of this section, you should have a solid understanding of how to use type conversion in your JavaScript code to create more powerful and flexible applications.

    Here's an example of type conversion in JavaScript:

              let numString = "42";
          let num = Number(numString);
          console.log(num); // Output: 42 (a number)

    In this example, we have a string that contains the value "42". We want to convert this value to a number so we can perform mathematical operations on it. To do this, we use the Number() function to convert the string to a number. We then store the result in a variable called num. Finally, we use console.log() to output the value of num, which is 42.

    Numbers & Math

    In JavaScript, numbers are used to represent numeric values, such as integers and decimals, and math operations can be performed on them using various built-in functions and operators. Understanding how to work with numbers and perform mathematical operations is crucial to building effective JavaScript applications. In this section, we will explore the basics of working with numbers in JavaScript, including number literals, mathematical operators, and common math functions. By the end of this section, you should have a solid understanding of how to use numbers and math functions in your JavaScript code.

    Here is an example of a math operation we can do:

              let num1 = 10;
          let num2 = 5;
          let sum = num1 + num2;
          let difference = num1 - num2;
          let product = num1 * num2;
          let quotient = num1 / num2;
          console.log(sum);        // Output: 15
          console.log(difference);  // Output: 5
          console.log(product);    // Output: 50
          console.log(quotient);   // Output: 2
